## MKH 是什么

MKH 是一款为中小型团队而生的基于.Net6 + Vue3 开发的业务模块化快速开发解决方案，目的是能够让中小团队快速搭建公司内部开发平台。它的前身是 [NetModular](https://github.com/iamoldli/NetModular)
