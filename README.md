## 快速开始

安装脚手架MKH CLI：

````bash
> dotnet tool install --global Mkh.Tools.CLI
````

更新脚手架MKH CLI：

````bash
> dotnet tool update --global Mkh.Tools.CLI
````

创建一个模块：

````bash
> mkh new
````
